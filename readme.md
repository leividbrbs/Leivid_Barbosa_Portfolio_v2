# leivid barbosa portfolio

a personal portfolio website built using html, css, and javascript. the design is inspired by letterboxd, spotify, and last.fm while showcasing my projects, background, and interests.

## about

i'm david leimar "leivid" barbosa, a second-year bs information technology student from tc07. i'm also an artist connection exude dancer and enjoy creating websites inspired by music, films, and visual design.

## features

- responsive layout
- about section
- skills section
- featured projects
- education and experience
- contact page
- links to github, instagram, spotify, last.fm, and letterboxd
- custom design without templates or frameworks
